## Docs

## Running the app

```bash
# development
$ task run && task logs:app
```

```bash
# restart dev environment
$ task restart && task logs:app
```

## Execute tests

```bash
# development
$ task test
```

```bash
# development
$ task test:ci
```

```bash
# development
$ task coverage
```

## Execute tests locally

```bash
# development
$ task test:local
```

## Stop containers

```bash
$ task stop
```
